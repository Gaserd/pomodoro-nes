Pomodoro Timer

About

A desktop Pomodoro timer with retro NES UI, click/finish sounds, and daily coin rewards.

  • Work/Break durations (25/5 by default), Start/Pause/Skip/Reset
  • Frameless window with a custom close button
  • Random finish sound and per-button click sounds
  • Coins earned after each Work interval, daily stats saved locally
Readme NES.css

Getting Started

Install dependencies and run in development:

npm install
npm start

Build Windows installer and portable:

npm run dist
